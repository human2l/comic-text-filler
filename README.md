# Comic Text Filler

A lightweight, automated local web tool designed to flawlessly overlay AI-generated structural dialogue (JSON) onto AI-generated blank comic images, creating full, retina-quality styled comic panels. 

## Vision

This tool replaces complex, manual typesetting with a fast, intuitive "drag, drop, and export" workflow. It leverages HTML5 Canvas (via Konva.js) to bring structured LLM script outputs directly onto blank comic visuals generated by visual diffusion models, bypassing the text-rendering limitations of image AI.

## Workflow

1. **AI Image Generation**: Create your comic panels (1-3 panes) using your preferred text-to-image AI (e.g., Gemini / Imagen 3), focusing purely on visual composition and empty speech bubbles.
2. **AI Script Generation**: Generate dialogue in structured JSON formats using your LLM.
3. **Smart Typesetting**: Upload the image, paste the JSON, and intuitively arrange auto-wrapped, styled text over the speech bubbles.
4. **Hi-Res Export**: Export the final combined comic image for publishing, retaining perfect character consistency and zero text artifacts.
